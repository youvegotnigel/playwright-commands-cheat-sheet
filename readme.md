# Playwright Cheat Sheet

A simple, interactive Playwright cheat sheet built with plain HTML, CSS, and JavaScript.

## Features

* Clickable command tiles
* Code examples with copy support
* Search functionality
* Color-coded categories
* Works offline

## How to Use

1. Open `index.html` in your browser
2. Click any command to see example
3. Use search to filter
